# Notice

This repository is a sanitized, standalone extraction of an ultratrace
microbubble ULM workflow. It keeps only the self-contained pieces: MACH
beamforming from a neutral demodulated-IQ ultratrace, SVD clutter filtering,
3D detection and localization, track linking and smoothing, compact binary
track export, and the browser viewers.

Device acquisition code, vendor raw-frame decoding, and any components outside
the standalone ULM scope are intentionally excluded. Original authorship and
dates of the ULM work are preserved in the Git history.
